SavePath is a modern personal finance web application designed to help users **track expenses, set financial goals, monitor progress, and visualize their budget** in one central hub. Built with clean HTML, CSS, and JavaScript, SavePath emphasizes clarity, usability, and a futuristic design aesthetic.

---

## Features

- **Dashboard Overview**
  - Snapshot of upcoming payments and goal progress
  - Clean grid layout with responsive design

- **Goals Page**
  - Create and track financial goals
  - Inline validation for numeric fields (e.g., "Must be a digit")
  - LocalStorage integration for saving goals

- **Quick Add**
  - Add one-time expenses or income
  - Add recurring payments (weekly/monthly)
  - Smooth card transitions for form selection

- **Calendar**
  - Organize bills, recurring payments, and financial events
  - Visual timeline for planning ahead

- **Visuals**
  - Charts and graphs to analyze spending and savings trends

- **Product Page**
  - SavePath-branded merch (shirts, mugs, hoodies) with empowering slogans
